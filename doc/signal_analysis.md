## 模块

### 检查典型性`check_typicality.m`

```matlab
1. plate #1
  - 回波的整体的标准差是 10.21。（所有帧、所有位置）
  - 而同一帧中，（不同位置）回波的标准差平均只有 5.47，占 0.5%。
2. plate #2
  - 回波的整体的标准差是 10.21。（所有帧、所有位置）
  - 而同一帧中，（不同位置）回波的标准差平均只有 5.78，占 0.6%。
```
